## Cara Install & Menjalankan 

### **Git Clone Repository**
```Sh
git clone https://github.com/rizzkyofc/Base-Bott-Whatsapp
cd Base-Bott-Whatsapp
```

### **Install Dependencies**
```Sh
npm install
yarn install
```

### **Menjalankan Bot**
```Sh
npm start
```
